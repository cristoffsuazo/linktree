# 🔗 Linktree Personal — Cristopher Suazo

Página de enlaces personales estilo Linktree, construida con HTML, CSS y Font Awesome. Diseño semiformal con estética oscura y animaciones elegantes.

---

## 📁 Estructura del proyecto

```
linktree/
├── index.html       # Estructura principal
├── style.css        # Estilos y animaciones
├── img/
│   └── Photo.jpeg   # Foto de perfil
└── README.md
```

---

## ✨ Novedades en v1.2

- Tipografía **Cormorant Garamond** + **DM Sans** (sin dependencia de Bootstrap)
- Fondo con **orbs animados** en gradiente dorado/azul/morado
- **Anillo giratorio** alrededor de la foto de perfil
- Tarjetas de links con efecto **glassmorphism** y hover interactivo
- Animaciones de entrada **escalonadas** por card
- Footer actualizado a **© 2026**
- Paleta de color oscura y semiformal (`#080c14`, dorado `#c8a96e`, azul `#7eb8d4`)

---

## 🛠️ Tecnologías

- HTML5
- CSS3 (variables, animaciones, backdrop-filter)
- [Font Awesome 6.5](https://fontawesome.com/) — íconos
- [Google Fonts](https://fonts.google.com/) — Cormorant Garamond · DM Sans

---

## 📦 Uso local

1. Clona el repositorio:
   ```bash
   git clone https://github.com/cristoffsuazo/linktree.git
   cd linktree
   ```

2. Para ver la versión más reciente (v1.2):
   ```bash
   git checkout v1.2
   ```

3. Abre `index.html` en tu navegador.
